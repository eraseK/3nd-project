# 3nd-TeamProject

## 주제 및 선정이유
  군중 또는 다수를 의미하는 CROWD + 자금을 의미하는 FUNDING
  - 단순히 물건을 사고 파는 쇼핑몰에 그치는 것이 아니라 개인의 투자와 소비 트렌드 파악 및 반영
  - 팔로잉, 메세지, 창작자에게 문의하기 등과 같은 상호작용 요소 多
  - JAVA,SPRING, DB, CSS, HTML 등의 기술적극 활용가능
